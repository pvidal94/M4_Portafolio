# Sistema de Gestión Municipal - Implementación Inicial en Python

Este repositorio contiene la implementación inicial del script `analisis_presupuestario_v1.py` para el portafolio profesional de Patricia Vidal Uribe.

## Objetivo
El script demuestra el dominio de los conceptos fundamentales de Python: variables, operadores, estructuras de control (`if/else`, `for`), estructuras de datos (`listas`, `diccionarios`) y modularización con `funciones`, aplicados a la **gestión presupuestaria municipal**.

## Programa: Análisis Presupuestario
Simula la carga de datos de proyectos y realiza cálculos básicos de presupuesto total y gasto acumulado, permitiendo un análisis inicial basado en condiciones y bucles.
